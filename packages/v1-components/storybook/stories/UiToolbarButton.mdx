# UiToolbarButton

Повторяет функционал стандартной кнопки `UiButton` с некоторыми ограничениями:

* `appearance` &ndash; значение принудительно установлено в `secondary` и не может быть изменено;
* `size` &ndash; значение принудительно устанавливается на стороне CRM в зависимости от того, где расширение запускается;
* `type` &ndash; значение принудительно установлено в `button` и не может быть изменено;

Данный тип кнопки нужен для корректной стилизации элементов, используемых на основном слое интерфейса,
элементов, размещенных не внутри некоторого модального элемента, вроде `UiModalSidebar` или `UiModalWindow`
