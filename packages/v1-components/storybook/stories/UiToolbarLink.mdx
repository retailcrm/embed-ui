# UiToolbarLink

Повторяет функционал стандартной ссылки `UiLink` с некоторыми ограничениями:

* `size` &ndash; значение принудительно устанавливается на стороне CRM в зависимости от того, где расширение запускается;

Данный тип ссылки нужен для корректной стилизации элементов, используемых на основном слое интерфейса, элементов,
размещенных не внутри некоторого модального элемента, вроде `UiModalSidebar` или `UiModalWindow`
