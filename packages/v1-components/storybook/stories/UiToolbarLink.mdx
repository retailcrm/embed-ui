import ToReact from '../ToReact.ts'

import UiToolbarLink from './UiToolbarLink.example.vue'

# UiToolbarLink

Является аналогом стандартной ссылки с узконаправленным применением. Исключением является предустановленный
размерный токен — `small`. В связи чем отсутствует свойство для настройки размера шрифта ссылки и иконок.


## API

### Типы ссылок

В зависимости от применения ссылок на форме выделены следующие типы: `breadcrumbs`, `default`, `navigation`,
`navigation-anchor` и `title`. Предустановленный тип для ссылок — `default`.

<p style={{ display: 'flex', columnGap: '16px' }}>
    <ToReact is={UiToolbarLink} text="breadcrumbs" appearance="breadcrumbs" />
    <ToReact is={UiToolbarLink} text="default" />
    <ToReact is={UiToolbarLink} text="navigation" appearance="navigation" />
    <ToReact is={UiToolbarLink} text="navigation-anchor" appearance="navigation-anchor" />
    <ToReact is={UiToolbarLink} text="title" appearance="title" />
</p>


### Переполненный контент

Если включено свойство `ellipsis`, то при переполнении текстового содержимого, оно будет обрезаться через многоточие. Иначе текст
переносится на следующую строку.

<p style={{ display: 'flex', flexDirection: 'column', gap: '8px', paddingLeft: '16px', maxWidth: '130px' }}>
    <ToReact is={UiToolbarLink} text="Long error message" external />
    <ToReact is={UiToolbarLink} text="Long error message" ellipsis external />
</p>


### Переход на новую вкладку

При необходимости по клику открывать ссылку в новой вкладке необходимо включить свойство `external`. Если установлен
компонент UiIcon или задан слот `icon`, то справа от текста ссылки также появится иконка.


### Дополнительная настройка внешнего вида

Для дополнительной настройки визуализации ссылки предоставлены свойства:
<ul>
    <li>
        <p style={{ display: 'inline', marginRight: '4px' }}>`accent` — добавляет тексту ссылки жирное начертание. Например,</p>
        <ToReact is={UiToolbarLink} tag="span" text="Accent link" accent external />
    </li>

    <li>
        <p style={{ display: 'inline', marginRight: '4px' }}>`dotted` — добавляет подчёркивание тексту ссылки. Например,</p>
        <ToReact is={UiToolbarLink} tag="span" text="Dotted link" dotted external />
    </li>

    <li style={{ background: 'rgb(105, 145, 209)', width: 'fit-content' }}>
        <p style={{ display: 'inline', marginRight: '4px' }}>`light` — инвертирует цвет ссылки и иконки в белый. Например,</p>

        <ToReact
            is={UiToolbarLink}
            tag="span"
            text="Accent link"
            light
            external
            ellipsis
        />

        <p style={{ display: 'inline', marginLeft: '4px' }}>Фон добавлен для демонстрации примера.</p>
    </li>
</ul>


### Слоты

В рамках компонента предоставлено два слота:
* `default` — текст ссылки;
* `icon` — иконка справа от текста ссылки.
