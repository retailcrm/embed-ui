# UiRadio

Одиночный выбор из представленных вариантов

## Описание

Используется для выбора одного варианта из набора. Состояние с галочкой — функция включена/выделена.
Без — выключена/не выделена.

### Accessibility

- Для набора радио-кнопок обязательно используйте общий `name`.
- Для группировки и названия набора используйте `fieldset` + `legend` (или `role="radiogroup"` с доступным именем).
- Для каждой отдельной кнопки нужен текстовый label через `<label for="...">` или `aria-label`/`aria-labelledby`.
