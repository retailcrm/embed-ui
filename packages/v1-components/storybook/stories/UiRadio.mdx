# UiRadio

`UiRadio` - компонент для выбора одного значения из набора взаимоисключающих вариантов.

## Когда использовать

* выбор одного варианта доставки, статуса, тарифа;
* переключение между взаимоисключающими режимами;
* формы, где важно явно показать все доступные варианты.

## Импорт

```ts
import { UiRadio } from '@retailcrm/embed-ui-v1-components/remote'
```

## Базовое использование

```vue
<template>
  <UiRadio id="delivery-courier" v-model:model="deliveryType" value="courier" name="delivery-type" />
  <label for="delivery-courier">Курьер</label>
</template>

<script setup lang="ts">
import { ref } from 'vue'
import { UiRadio } from '@retailcrm/embed-ui-v1-components/remote'

const deliveryType = ref('courier')
</script>
```

## Сценарии

### Группа с общим `name`

Чтобы радио-кнопки работали как единая группа, у них должен быть одинаковый `name`.

### Логика выбора

Компонент считается выбранным, когда `model === value`.
При выборе `UiRadio` эмитит:

* `change` - выбранное значение;
* `update:model` - выбранное значение для `v-model:model`.

## A11y

* Для группы радио-кнопок используйте общий `name`.
* Для подписи группы предпочтительно использовать `fieldset` + `legend`.
* Для каждой кнопки задавайте текстовую метку через `<label for="...">` (или `aria-label`/`aria-labelledby`).
