# UiModalSidebar

Выдвигаемая боковая панель для размещения дополнительных настроек или информации

## API

### Виды панелей

Стандартная панель выводится справа и бывает двух размеров: `sm — 416 px` и `lg — 720 px`. Выбор размера определяется
типом его контента. Это могут быть дополнительные настройки для параметров на странице, а также информационный материал
по новым сущностям.

<div style={{ display: 'flex', columnGap: '16px' }}>
    <ToReact is={UiModalSidebar} buttonTitle="Стандартная панель (sm)" />
    <ToReact is={UiModalSidebar} buttonTitle="Стандартная панель (lg)" size="lg" />
</div>

Профиль — выводится слева, по клику на иконку профиля. На панели показаны данные пользователя, тариф системы и
ссылки на настройки.

<ToReact is={UiModalSidebar} buttonTitle="Левая панель (sm)" direction="left" />


### Управление состоянием

Панель можно закрыть:
* нажатием на иконку закрытия панели, либо клавишей `Esc`;
* смена состояния флага `opened`;
* кликом вне области панели, если панель является блокирующей. Такая панель имеет затемнённую подложку.

При необходимости можно запретить закрывать панель вышеперечисленными способами, если выключить флаг `closable`. Тогда
панель будет оставаться в открытом состоянии до тех пор, пока не станет вновь закрываемой.

Для настройки времени удаления плавающего элемента со страницы, если он был скрыт, используется свойство `disposeTimeout`.
Значение задаётся в миллисекундах, поддерживаются как числа, так и строковые форматы.


### Прокрутка

При прокрутке шапка и футер закреплены и прокручивается только контент в теле панели. Если панель не блокирующая, то есть
для неё не установлен флаг `fixed`, то можно прокручивать и панель, и страницу по отдельности в зависимости от положения
курсора.

<ToReact is={UiModalSidebar} buttonTitle="Фиксированная панель (sm)" fixed />

По-умолчанию, содержимое тела панели помещается в контейнер со скроллом. Данное состояние определяется свойством
`scrolling` со значением `normal`. Вид полосы прокрутки можно поменять на стандартный, предоставляемый браузерами.
Для этого необходимо установить значение `native`.

Чтобы отключить прокрутку для контента в теле панели не обходимо установить значение `none`.


### Информационная панель

Панель с пояснениями для настроек. Не блокируют контент, чтобы пользователь одновременно изучал материал и применял
полученные знания на странице. В таких панелях могут быть тексты, элементы интерфейса и скриншоты.


### Вложенность

В качестве дочерних модальных окон можно использовать и другие модальные элементы на основе `UiModal`, например,
`UiModalWindow`

<ToReact is={UiModalSidebar} buttonTitle="Панель со вложенной шторкой (sm)" hasNesting={true} footer />


## Слоты

Для настройки внешнего вида заголовка в шапке панели содержится слот `title`. Он позволяет добавлять опционально необходимые
компоненты. Например, если в панели находится многоуровневая настройка, то перед заголовком может быть добавлена иконка для
возврата назад.

Слот `default` позволяет передать содержимое тела панели.

Футер в панели отображается, если определён слот `footer`. Чаще всего в нём содержатся кнопки сохранения, отмены и третичные
кнопки копирования и удаления.

<ToReact is={UiModalSidebar} buttonTitle="Панель с заданными слотами" footer />


## События

Компонент генерирует события:
* `open` — при открытии панели;
* `close` — при закрытии панели. Содержит данные о методе закрытия, а именно при клике по кнопке закрытия (`Cross`),
при клике на область вне панели (`Outside`) или по событию esc (`Esc`);
* `close-cancel`— при попытке закрыть панель, если флаг `closable` выключен. Содержит данные о методе закрытия;
* `toggle` — при раскрытии и закрытии панели. Содержит актуальное значение флага `opened`;
* `shown` — при завершении раскрытия;
* `showing` — в начале раскрытия;
* `hidden` — при завершении сокрытия;
* `hiding` — в начале сокрытия;
* `update:opened` — при изменении значения флага `opened`;
* `update:overlapped` — при изменении состояния перекрытия;
* `scroll:y:end` — при прокрутке содержимого в самый конец по y-оси.
