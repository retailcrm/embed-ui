import ToReact from '../ToReact.ts'

import UiError from './UiError.example.vue'

# UiError

Компонент визуализации некоторой ошибки

## API

### Контент

Передачу текстового сообщения ошибки можно осуществлять через свойство `message`
или слот по-умолчанию (для передачи HTML разметки)

<p className="flex-column pl-4 gap-2">
    <ToReact is={UiError} message="Error message" />
    <ToReact is={UiError} useSlot />
</p>


### Выравнивание содержимого

В рамках компонента допускается выравнивание текста по левому или правому краю.

<p className="flex-column pl-4 gap-2">
    <ToReact is={UiError} message="Error text on the left" />
    <ToReact is={UiError} message="Error text on the right" align="right" />
</p>


### Переполнение

Если включено свойство `ellipsis`, то при переполнении текстового содержимого, оно будет обрезаться через многоточие.
Без флага `ellipsis` контент переносится на следующую строку

<p className="flex-column pl-4 gap-2" style={{ width: '100px' }}>
    <ToReact is={UiError} message="Long error message" />
    <ToReact is={UiError} message="Long error message" ellipsis />
</p>
