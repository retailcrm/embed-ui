# UiCopyButton

`UiCopyButton` - кнопка для копирования значения в буфер обмена с подсказкой об успешном копировании.

## Когда использовать

* копирование ссылки, id, токена, API-ключа;
* быстрое копирование значений из карточек и таблиц;
* интерфейсы, где нужна явная индикация «Скопировано».

## Импорт

```ts
import { UiCopyButton } from '@retailcrm/embed-ui-v1-components/remote'
```

## Базовое использование

```vue
<template>
  <UiCopyButton text="Текст для копирования" size="sm">
    <template #hint>
      Скопировать
    </template>

    <template #hint-copied>
      Скопировано
    </template>
  </UiCopyButton>
</template>

<script setup lang="ts">
import { UiCopyButton } from '@retailcrm/embed-ui-v1-components/remote'
</script>
```

## Сценарии

### Кастомный триггер

Слот `trigger` позволяет подставить любую кнопку или элемент, который будет вызывать копирование.

### Настройка всплывающей подсказки

Через `tooltipOptions` можно передать параметры `UiTooltip` (например, `placement`, `offset`, `container`).

### Размер кнопки

Параметр `size` принимает размеры из `UiButton` (`xs`, `sm`, `md`, `lg`).

## Слоты

Компонент поддерживает:

* `trigger` - кастомный элемент-триггер;
* `hint` - текст подсказки до копирования;
* `hint-copied` - текст после успешного копирования.

## События

* `error` - эмитится, если не удалось записать текст в буфер обмена.

## A11y

* По умолчанию используется нативная кнопка `UiButton`, доступная с клавиатуры.
* Статус копирования сообщается через содержимое `UiTooltip`.
* При недоступном Clipboard API используется fallback через скрытое поле ввода.
