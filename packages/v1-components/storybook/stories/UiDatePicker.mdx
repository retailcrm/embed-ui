# UiDatePicker

`UiDatePicker` - компонент выбора даты и диапазона дат поверх `UiCalendar` с полем ввода и popper-контейнером.

## Когда использовать

* фильтры по дате/периоду в embedded-интерфейсах;
* формы с выбором одной даты или диапазона;
* сценарии с быстрыми пресетами (`quickOptions`);
* формы, где нужно подтвердить выбор через `Apply`.

## Импорт

```ts
import { UiDatePicker } from '@retailcrm/embed-ui-v1-components/remote'
```

## Базовое использование

```vue
<template>
  <UiDatePicker v-model:value="value" />
</template>

<script setup lang="ts">
import { ref } from 'vue'
import { UiDatePicker } from '@retailcrm/embed-ui-v1-components/remote'

const value = ref<Date | null>(new Date())
</script>
```

<ToReact is={UiDatePickerBasic} />

## Модель данных

Тип значения зависит от `type`:

* `type="single"` -> `Date | null`;
* `type="range"` -> `Date[] | null`.

События:

* `update:value` - событие для `v-model:value`;
* `change` - пользовательское изменение значения;
* `open` / `close` - открытие и закрытие popper.

## Режимы синхронизации

`synchronization="instant"`:

* в `single` значение эмитится сразу после выбора даты;
* в `range` эмит после выбора второй границы.

`synchronization="confirmed"`:

* выбор хранится во внутреннем состоянии;
* для публикации значения используйте `Apply`;
* `Clear` очищает выбранное значение.

## A11y и клавиатура

Компонент использует паттерн `combobox` + `dialog` + `grid`:

* `role="combobox"` на корневом элементе;
* `role="dialog"` на popup;
* внутри popup используется `UiCalendar` с семантикой `grid`.

Клавиатура в textbox:

* `ArrowDown` / `ArrowUp` - открыть popup;
* `Enter` - открыть popup; в `confirmed` при открытом popup эквивалент кнопке `Apply`;
* `Escape` - закрыть popup.

После открытия календаря доступны клавиши навигации `UiCalendar` (`Arrow`, `Home/End`, `PageUp/PageDown`, `Enter/Space`).

## Ограничения текущей версии

Вложенный выбор времени внутри `UiDatePicker` в текущей версии не поддерживается.
Для выбора времени используйте отдельный `UiTimePicker`.
