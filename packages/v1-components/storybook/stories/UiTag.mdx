# UiTag

Тег, компонент визуального обозначения состояния

## API

### Размеры

Для тегов предусмотрены токены размеров: `lg`, `md`,`sm` и `xs`. Представленные размеры влияют
на высоту, отступы, размер контента и иконок.

<p style={{ display: 'flex', alignItems: 'center', gap: '16px', paddingLeft: '8px' }}>
    <ToReact is={UiTag} text="lg" pinned removable />
    <ToReact is={UiTag} text="md" size="md" pinned removable />
    <ToReact is={UiTag} text="sm" size="sm" pinned removable />
    <ToReact is={UiTag} text="xs" size="xs" pinned removable />
</p>

### Настраиваемый внешний вид

Изменение внешнего вида элемента осуществляется настройкой следующих свойств:
* `background` — фоновый цвет тега. Задавать цвет можно с использованием любой доступной цветовой модели;
* `saturated` — инверсия цвета текста и иконки в белой цвет;
* `ticker` — анимация бегущей строки при наведении;
* `interactive` — курсор и анимация при наведении.

<p style={{

## События

Компонент предоставляет нативные события `click`, `focus` и `blur`. Если для тега установлено свойство `removable`, то
клик по иконке удаления вызывает появление события `remove`.

## Слоты

Для адаптации компонента под более широкий спектр применения выделены слоты:
* `icon` — иконка слева от основного контента тега;
* `default` — основной контент тега;
* `remove-icon` — иконка удаления, расположенная справа от основного контента.
