# UiTooltip

Всплывающая подсказка

## Поведение

По-умолчанию подсказка отображается по наведению на целевой элемент и скрывается если увести курсор с элемента
или кликнуть по нему.

## Применение

Компоненту `UiPopper`, используемому в качестве базы для `UiTooltip`, требуется иметь информацию о цели.
Встроенные компоненты `embed-ui` такую информацию предоставляют, но чтобы связать её с плавающим элементом, нужно
использовать перемычку `UiPopperConnector`:

```html
<template>
    <UiPopperConnector>
        <UiButton disabled>
            Сохранить
        </UiButton>

        <UiTooltip>
            Перед сохранением нужно указать все обязательные поля
        </UiTooltip>
    </UiPopperConnector>
</template>
```
